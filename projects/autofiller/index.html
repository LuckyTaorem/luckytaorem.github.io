📌 Steps to Install

  1. 1️⃣ Download the ZIP file from the link above.
  2. 2️⃣ Extract the ZIP file to a folder.
  3. 3️⃣ Open Chrome or Edge.
  4. 4️⃣ Go to Extensions (chrome://extensions/ or edge://extensions/).
  5. 5️⃣ Enable Developer Mode (toggle in the top right).
  6. 6️⃣ Click Load Unpacked.
  7. 7️⃣ Select the extracted folder and click OK.
  8. ✅ Done! The extension is now installed.
